# House Price Prediction - Linear Regression

This project uses Linear Regression to predict house prices based on:

- Carpet Area
- Floor
- Bathroom

## Tools Used
- Python
- Pandas
- Scikit-learn
- Google Colab

## Files
- mlproject.ipynb → Main Notebook
- house_small.csv → Dataset (Reduced Size)

## Models Used
- Linear Regression
- Multiple Linear Regression
- Polynomial Regression
- Lasso Regression
